Reload

Table of Contents

  • Overview
  • Syntax
Note:This task is applicable only to Zoho Creator.

Overview

The Reload deluge task reloads the form.

Note: When a form is reloaded, its unsubmitted field entries (if any) will be lost.

Syntax

reload;

This task can be used in the following events

When a record is Created
On Load Yes
On Validate No
On Success No
On User input Yes
Subform on add row No
Subform on delete row No
Subform field on User input No
When a record is Created or Edited
On Load Yes
On Validate No
On Success No
On User input Yes
Subform on add row No
Subform on delete row No
Subform field on User input No
When a record is Edited
On Load Yes
On Validate No
On Success No
On User input Yes
On Update Yes
Subform on add row No
Subform on delete row No
Subform field on User input No
When a record is Deleted
On Validate No
On Success No
Other workflow events
On a scheduled date No
During approval process No
During payment process No
In a Custom Function No
In an Action item in report No
