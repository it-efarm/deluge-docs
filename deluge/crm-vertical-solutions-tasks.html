Zoho CRM Vertical Solutions Integration Tasks

  • Get User Data: Fetches the information about the users of an extension or vertical solution
  • Invoke Connector Task: Invokes a connector associated with an extension or vertical solution
  • Set Organization Variable: Assigns value to a custom variable
  • Get Organization Variable: Obtains the value of a custom variable
  • Get Module Metadata: Gets a particular module's meta data
  • Get All Module Metadata: Obtains the meta data of all the different modules/li>
  • Get Organization Information: Retrieves organization information
