Overview

The len() function takes a string as an argument. It returns the count of characters (including spaces) in that string.

Return Type

  • Number

Syntax

<variable> = <string>.len();

(OR)

<variable> = len(<string>);

(OR)

<variable> = <string>.length();

(OR)

<variable> = length(<string>);
Parameter Description Data type
<variable> Variable which will contain the count of characters NUMBER
<string> The string whose number of characters will be returned TEXT

Examples

text = "Create database applications with Zoho Creator";
count= text.len();      //returns 46
