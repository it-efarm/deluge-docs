Overview

The toXml function takes a JSON/MAP formatted text as an argument, and returns it as an XML.

Return Type

  • TEXT

Syntax

<text>.toXml();

where,

Parameter Data type Description
<variable> TEXT Variable which will contain the converted XML.
<text> TEXT The JSON formatted text which needs to be converted to XML.

Examples

 jsonText = "{Name:'Lia Shelton', OrderNo:[343,443,577]}";    
 info jsonText.toXML(); 
 /* returns the following XML
 <root>
 <OrderNo>343</OrderNo>
 <OrderNo>443</OrderNo>
 <OrderNo>577</OrderNo>
 <Name>Lia Shelton</Name>
 </root>
 */
