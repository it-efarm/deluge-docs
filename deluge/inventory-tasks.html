Zoho Inventory Integration Tasks

  • Get organization: Fetches the associated organizations
  • Get Records: Fetches records from the specified module
  • Get Records By ID: Fetches the details of the specified record
  • Create Record: Creates a record
  • Update Record: Updates the specified record
  • Mark Status: Changes the status field with one of the allowed values
