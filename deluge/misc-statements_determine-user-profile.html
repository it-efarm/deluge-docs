Determine profile of logged-in app user

Table of Contents

  • Overview
  • Return
  • Syntax
  • Example

Note: This task is applicable only to Zoho Creator.

Overview

This deluge task is used to check if the logged-in app user belongs to a specified profile or not. 

Return

This task returns boolean value 'True' if the logged-in user belongs to the specified profile. Otherwise, it returns 'False'.

Syntax

<variable> = thisapp.permissions.isUserInProfile( <Profile_Name> );

Param Explanation

<variable>

(optional)

Variable which will hold the returned value.
<Profile_Name>

Name of the profile to check if the logged-in user has been assigned to that profile.

Points to note for this parameter:

  • If the specified profile name does not exist, execution will be stopped during run-time and an error will be displayed.
  • You can view the existing profiles in your app from the Permissions page.

Data type is Text.

Example

A sample snippet being used to determine if the logged-in user belongs to the "Read" profile.
