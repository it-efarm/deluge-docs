Get template by ID from Zoho Sign

Table of Contents

  • Overview
  • Syntax
  • Example
  • Response Format
  • Related Links

Overview

You can use the zoho.sign.getTemplateById task to fetch templates based on ID from Zoho Sign.

Syntax

<response> = zoho.sign.getTemplateById(<template_ID>, [<connection>]);

where,

Params Data Type Description

<response>

KEY-VALUE

The details of the template image as TEXT and details of the fields being used in the template - their type, their size, and other relevant information.

<template_ID>

NUMBER

The ID of the template that needs to be fetched.

Note: You can get the ID of a Zoho Sign template from its URL. The URL is in the format: https://sign.zoho.com/zs#/template/viewer/<template_id>
(or)
You can get the template ID from the response of zoho.sign.getTemplate task. Click here to learn how to fetch the template ID.

<connection>

(optional)*

 TEXT

The name of the connection.

*Note: This param is not applicable to Zoho Creator and mandatory in Zoho Cliq.

Example: Fetch templates from Zoho Sign based on ID

The following script fetches a template from Zoho Sign based on the specified template ID. 

response  = zoho.sign.getTemplateById(32076000000002011);

where:

response
The KEY-VALUE response that represents the details of the template image, fields used in the template, and other relevant information. 
32076000000002011
The NUMBER that represents the ID of the template that needs to be fetched.
