Description

The zoho.writer.documents.setDescription task adds or updates your document with the description you provide. This task is based on the Zoho Writer Add or Update Description API.

Note: Applicable to all services except Zoho Creator.

Syntax

<response> = zoho.writer.documents.setDescription(<document_id>, <Description>, <connection>);

where:

Params Data type Description
<response> KEY-VALUE The response returned by Zoho Writer. It reflects the status of the addition/update to the document's description.
<document_id> TEXT

The ID of the document for which the description needs to be added/updated.

The document ID can be obtained from the URL of the document in which the description needs to be added or updated.

The URL is in the following format: https://writer.zoho.com/writer/open/<document_id>

<Description> TEXT

Pass a string value for a description.
<connection> TEXT

The connection's link name.

Note:

  • To prevent new authtoken generation and ensure new integration tasks (created after the deadline specified in the post) work as expected, a Zoho OAuth connection with appropriate scopes is mandatory. Existing integration tasks will continue to work, with or without the connections parameter, until you manually delete the authtoken from Zoho Accounts.

  • Add relevant scopes from the Zoho Writer API when creating the connection.

  • Refer to this post for a list of Zoho services that support the Connections page.

  • Learn more about connections

Example

The following script updates the description of the given Writer document. 

document_id = "h36bc2a6c50b2754a405880c1d1679a5f5be9";
Description = "Online Word Processor";
response = zoho.writer.documents.setDescription(document_id, Description, "writer_oauth_connection")
info response;

where:

response
The KEY-VALUE response returned by Zoho Writer. It represents whether the document description has been added or updated. 
"h36bc2a6c50b2754a405880c1d1679a5f5be9"
The TEXT that represents the ID of the document in Zoho Writer. 
"Online Word Processor"
The TEXT that represents the new description of the document. 
"writer_oauth_connection"
The TEXT that represents the name of the connection.

Response Format

{
  "result": "success",
  "description": "Online Word Processor",
  "message": "Document description has been updated successfully."
}
