Get Documents from Zoho Writer

Table of Contents

  • Description
  • Syntax
  • Example
  • Response Format
  • Related Links

Description

The zoho.writer.getDocuments task is used to fetch a list of all the documents from Zoho Writer. This integration task is based on the Zoho Writer API - Get List of Documents with Specified Parameters.

Note: On average, it takes about a minute to index a file/folder after it is created or updated. This indexing time can increase if the queue size increases. You will not get the expected result if you execute zoho.writer.getDocuments task before the document you need to fetch is indexed.

Syntax

<response> = zoho.writer.getDocuments(<category>, <offset>, <limit>, <sort_by>, <connection>);
where:
Params Data type Description

<response>

KEY-VALUE

The response returned by Zoho Writer. It represents the number of documents fetched and their metadata.

<category>

TEXT

The category based on which the documents are filtered and fetched.

The possible values for the parameter - category are as follows:

  • all - Fetches a list of all documents
  • shared_to_me - Fetches a list of documents which are shared to you
  • owned_by_me - Fetches a list of documents which are created by you

<offset>

NUMBER

The start index from which the documents will be fetched. The index value starts from 0.

<limit>

NUMBER

The number of documents that will be fetched.

Note: The maximum number of documents that can be fetched is 100.

<sort_by>

TEXT

The order in which the documents will be fetched.

The possible values for the parameter - sort_by are as follows:

  • created_time - Fetches the documents in the order in which they are created. The recently created document is fetched first.
  • modified_time - Fetches the documents in the order of the time they are modified. The recently modified document is fetched first.
  • lastopened_time - Fetches the documents in the order in which they were last opened.

<connection>

TEXT

The link name of the connection.

Note:

  • In view of stopping new authtoken generation, a Zoho OAuth connection with appropriate scopes is mandatory in order for new integration tasks (created after the deadline specified in the post) to work as expected. Existing integration tasks will continue to work with or without the connections parameter until you manually delete the authtoken from accounts.
  • Add relevant scopes mentioned in ​Zoho Writer API while creating the connection.
  • Refer to this post for the list of Zoho services that support the connections page.
  • Learn more about connections

Example

The following script fetches a list of first two recently created documents from Zoho Writer. 

response = zoho.writer.getDocuments("all",0,2,"created_time","writer_oauth_connection");

where:

response
The KEY-VALUE response returned by Zoho Writer. It represents the number of documents fetched and their respective metadata. 
0
The NUMBER that represents the start index from which the documents will be fetched. 
2
The NUMBER that represents the total number of documents that will be fetched. 
"created_time"
The TEXT that represents the sort_by value. Here, the documents are fetched based on the order of the time in which they are created; the recently created document is fetched first. 
"writer_oauth_connection"
The TEXT that represents the name of the connection.
