Get Sign Templates in Zoho Writer

Table of Contents

  • Description
  • Syntax
  • Example
  • Response Format
  • Related Links

Description

The zoho.writer.getSignTemplates lists all the sign templates created by the user. This is based on Zoho Writer Get sign templates API

Note: Applicable to all services except Zoho Creator.

Syntax

<response> = zoho.writer.getSignTemplates(<optional_settings>, <connection>);
Params Data type Description
<response> KEY-VALUE The response returned by Zoho Writer. It represents the list of all the sign templates and its metadata.
<optional_settings> KEYVALUE  The optional parameters you can send to fetch data according to your preference.
<connection> TEXT

The link name of the connection.

 

Note: 

 

  • To prevent new authtoken generation, and ensure new integration tasks (created after the deadline specified in the post)  work as expected, a Zoho OAuth connection with appropriate scopes is mandatory. Existing integration tasks will continue to work with or without the connections parameter until you manually delete the authtoken from accounts.

  • Add relevant scopes from the Zoho Writer API while creating the connection.

  • Refer to this post for the list of Zoho services that support the Connections page.

Learn more about connections

 

optional_settings

offset Integer

Set offset parameter to get the list of sign templates from a specified page.

By default, offset value will be 0.

limit Integer

Set offset parameter to get the list of sign templates from the respective page.

By default, offset value will be 10.

sortby String

created_time 
Get a list of sign templates sorted by created time.
modified_time
Get a list of sign templates sorted by last modified time.
name
Get a list of sign templates sorted by template name.
Defaultdateformat: IS08601

By default, templates will be sorted and listed based on the modified_time.

sort_order_by String

ascending
Get the list of sign templates in the ascending order based on the 'sortby' param value configured.
descending
Get the list of sign templates in the descending order based on the 'sortby' param value configured.

By default, the templates will be sorted and listed in the descending order.

category String

all
Get all the list of sign templates.
shared_to_me
Get only the list of sign templates which are shared to you.
owned_by_me
Get only the list of sign templates which are created by you.

By default, the category value will be all. Here, all the sign templates will be listed.

Example

The following script gets all the sign templates of the user in descending in the order of document modified time, 5 at a time 

optional_settings = Map();
optional_settings.put("offset", "0");
optional_settings.put("limit", "5");
optional_settings.put("sortby", "modified_time");
optional_settings.put("sort_order_by", "descending");
optional_settings.put("category", "all");
response = zoho.writer.getSignTemplates(optional_settings, "writer_oauth_connection");
info response;

where:

response
The KEY-VALUE response returned by Zoho Writer. It represents the list of sign templates for the user. 
0
The KEY-VALUEThe INTEGER which represents from which page to get the list of sign templates. 
5
The KEY-VALUEThe INTEGER which represents how many sign templates to get in a single call. 
modified_time
The KEY-VALUE The String which represents the sortby order 
descending
The KEY-VALUEThe String which represents the display order 
all
The KEY-VALUE The String which represents to get all the sign templates for the user.
